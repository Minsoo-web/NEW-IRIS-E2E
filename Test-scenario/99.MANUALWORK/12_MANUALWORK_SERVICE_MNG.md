##### 무료체험 신청하기

- QA 계정(qa.mobigen@gmail.com / qa25251@)으로 로그인
- `'무료체험하기'` 버튼 클릭
- Analyzer / Studio를 선택하고 하단 도메인 정보 입력 후 중복확인 버튼 클릭
- `'무료체험 신청하기'` 버튼 클릭
- 구글메일로 root 계정 비밀번호 취득 확인

##### 무료체험 종료

- 계정메뉴 - 서비스 관리 선택
- 가장 최근에 신청한 서비스 도메인 존재확인
- 해당 카드의 `'무료체험 종료'` 버튼 클릭
- `'무료체험 종료'`, `'서비스 만료'` badge 확인
